
# Using MQTT to connect an IoT Hub with Python

## Setup
1. Open main.py and edit the following fields with the correct IoT Hub and device information
    * IOT_HUB_HOSTNAME
    * IOT_HUB_DEVICE_ID
    * IOT_HUB_SAS_KEY

2. Install Paho MQTT

```
pip install paho-mqtt
```

3. Run the script

```
python main.py
```
